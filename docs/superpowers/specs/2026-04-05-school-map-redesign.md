# BetterSchool 地图页面重新设计

**日期：** 2026-04-05  
**状态：** 已确认，待实现

---

## 背景与目标

BetterSchool 是一个面向澳大利亚家长的选校工具，核心功能是在地图上展示学校并支持筛选排序。当前数据包含 952 所学校（字段：`school_name`, `suburb`, `state`, `postcode`, `rank`, `sector`, `score`, `lat`, `lng`, `address_suburb`, `address_state`, `school_type`）。

**现存核心问题：**
- 地图和侧边栏列表完全割裂（地图显示阿德莱德，列表却显示维州学校）
- 点击列表卡片，地图没有反应；点击地图圆点，列表不滚动
- 所有圆点大小一致，无法从地图上感知学校质量差异
- 首页是一个摩擦点（欢迎页 → 再点击 → 才能看到地图）
- 地图被限制在容器内，没有充分利用屏幕空间

**设计目标：** 以地图为核心主视图，其他 UI 元素悬浮其上，列表由地图视口驱动，实现真正的地图优先体验。

---

## 核心决策

### 用户工作流
家长以**位置优先**的方式使用本工具：打开页面后看到自己所在城市的学校分布，在地图上浏览，再通过筛选缩小范围。不需要主动输入地址——IP 定位已经足够，家长关注的是城市/区域级别的范围，不需要精确到街道。

---

## 布局设计

### 全屏地图底层
- Leaflet 地图占满整个视口（移除原有的 flex 容器布局）
- IP 定位逻辑保持不变（页面加载后自动跳转到用户所在城市，zoom=10）
- TileLayer（OpenStreetMap）保持不变

### 悬浮层（`position: absolute` / `fixed`，覆盖在地图上）

```
┌─────────────────────────────────────────────┐
│  [顶部筛选栏 — Chip 标签]               top:12px │
```

## 组件设计

### 1. 顶部筛选栏

- 位置：`position: absolute; top: 12px; left: 12px; right: 12px`
- 样式：Chip 标签（圆角胶囊，选中态紫色 `#4f46e5`）
- 筛选项：
  - **办学性质**：All（默认）/ 政府 / 非政府
  - **最低评分**：All（默认）/ 80+ / 90+ / 95+
  - **学校类型**：All（默认）/ Primary / Combined
- **移除 State 筛选**：地图本身即地理筛选器，State 下拉冗余
- 筛选变更后：地图圆点和左侧列表同步更新

### 2. 地图圆点（CircleMarker）

- **大小**：`radius = 5 + (score - 60) / 40 * 13`，范围 5–18px（score 60→5px，score 100→18px）
- **颜色**：
  - 政府学校：`#22c55e`（绿色，保持不变）
  - 非政府学校：`#f97316`（橙色，保持不变）
- **选中状态**：填充色改为 `#4f46e5`（紫色），白色边框 2px，额外渲染一个同心大圆（`fillOpacity: 0.25, radius +8`）模拟发光效果
- **Hover 状态**：轻微放大（CSS transform scale 1.3），光标 pointer

### 3. 左侧列表面板

- 位置：`position: absolute; top: 56px; left: 12px; bottom: 12px; width: 280px`
- 背景：白色，圆角 12px，阴影
- **标题行**：「当前区域 X 所学校」（X 实时计算）+ 排序下拉（按排名 / 按评分）
- **列表更新触发**：监听地图 `moveend` 和 `zoomend` 事件，计算 `map.getBounds()` 内的学校
- **卡片内容**：学校名 + 区/州 + 评分徽章 + 排名 + 办学性质标签
- **选中状态**：紫色边框 + 浅紫背景 `#eef2ff`
- **收起/展开**：面板右边缘有 `‹` 箭头按钮；收起后面板隐藏，箭头变 `›`；可完全展开地图视野

### 4. 右侧详情面板

- 位置：`position: absolute; top: 56px; right: 12px; bottom: 12px; width: 240px`
- **默认隐藏**（`display: none` 或 `opacity: 0 + pointer-events: none`）
- **触发显示**：点击地图圆点 或 点击左侧列表卡片
- **关闭**：点击面板内 ✕ 按钮，或点击地图空白区域
- **内容**：
  - 学校全名
  - 区、州、邮编
  - 全国排名（`#N`）
  - 评分（大号数字徽章）
  - 办学性质标签
  - 学校类型（Primary / Combined）

### 5. 地图图例

- 位置：右下角，`position: absolute; bottom: 36px; right: 12px`（在 Leaflet 原生控件上方）
- 内容：
  - 🟢 政府学校  
  - 🟠 非政府学校  
  - ○ ◯ 〇 大小 = 评分高低

## 交互流程

### 主要场景 A：浏览地图，查找当前区域学校
1. 用户打开页面 → IP 定位跳转到所在城市（zoom 10）
2. 地图加载完成 → 计算视口内学校 → 左侧列表显示这些学校，按排名排序
3. 用户拖动/缩放地图 → `moveend`/`zoomend` 触发 → 列表实时更新
4. 用户调整顶部筛选 → 圆点和列表同步过滤

### 主要场景 B：从列表选中学校，查看详情
1. 用户点击左侧列表中的学校卡片
2. 地图跳转到该学校位置（`map.setView`）并高亮圆点（紫色 + 发光）
3. 右侧详情面板显示，展示该学校完整信息
4. 用户点击 ✕ 或地图空白区域 → 详情面板隐藏，圆点恢复原色

### 主要场景 C：从地图点击圆点
1. 用户点击地图圆点
2. 圆点变为选中状态（紫色 + 发光）
3. 左侧列表滚动到对应卡片并高亮
4. 右侧详情面板显示
